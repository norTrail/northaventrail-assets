# NorthavenTrail.org Website

## Project Overview
Static client-side files (JS, CSS, JSON) for northaventrail.org.
Files are served via Cloudflare CDN at assets.northaventrail.org.
The repo is connected to Cloudflare Pages — committing a file deploys it automatically.

## Pages That use these files
- /hawk-lights
- /where-to-park
- /trailmap
- https://northaventrail.org/report-trail-issue
- https://northaventrail.org/adoptgarden
- https://northaventrail.org/map-points-of-interest
- https://northaventrail.org/tails-2026 (which will move to https://northaventrail.org/tails when the project goes live)
- https://northaventrail.org/trail-captains
- https://northaventrail.org/valentine-cling-map-2027

The goal of these files is to create a common UX / UI for MapBox maps and mapping data across the website.  The data for these files (json files) is stored in Google Sheets and generated by Google App Script.  The goal is to move these files to GitHub and Cloudflare. 

## Important
- Do NOT run a dev server — these are static files only
- No build step required
- Changes go live when committed and pushed to GitHub
- JSON data files (trail-poi, trail-captains, sheep-locations, etc.) are generated by Google Apps Script — do NOT hand-edit them
- Breaking JS/CSS changes that need cache-busting require a new versioned filename (files are cached 1yr immutable)
